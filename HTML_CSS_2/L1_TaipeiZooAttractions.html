• 文湖線




    「台北市立動物園」位於台北市文山區萬興里，全園佔地面積計165公頃。
    1989年臺北市立動物園決定要搬遷到木柵園區，提供動物們更好、更舒適的環境，
    也漸漸的提升服務品質，陸續也推出許多寓教於樂的活動，更創新的舉辦認養活動，
    讓野生動物能獲得實質的保護，社會大眾也能參與保育計畫。

    並在2003年開始，建立了酷cool節能屋、濕地生態池、野生動物收容中心、池上牧場繁殖群等，
    讓民眾對於前來台北市立動物園旅遊的意願能夠造成更大的吸引力。
